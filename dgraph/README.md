# Dgraph

- ネイティブGraphQLデータベース
- 水平方向にスケーラブルな分散型データベース
- ACIDトランザクション
- 一貫性のあるレプリケーション
- 線形化可能な読み取り

## Starting

Create .env file, and edit dgraph docker container ports.

```shell
cp .env.example .env
vi .env
```

Create compose, and starting...

```shell
docker compose up -d
```

Stop compose

```shell
docker compose stop
```

Compose starting

```shell
docker compose start
```
